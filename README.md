### [`@nurav/envguard`](https://www.npmjs.com/package/@nurav/envguard) — the `.env` problem every team has, finally solved.

Validates, syncs, and strictly enforces environment variable schemas across **local dev, CI/CD pipelines, and production** — so missing keys never cause a 2am incident again.

```bash
npm i -g @nurav/envguard

envguard check           # Catch missing keys before they catch you
envguard sync            # Auto-fill keys from .env.example
envguard check --strict  # Fail CI if keys are missing
envguard check --json    # Structured output for automation
```
